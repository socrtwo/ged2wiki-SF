*** Gedcom2 Wiki Converter 1.0 ***

This program is supposed to convert gedcom files to proper wiki family tree template code as described here: http://en.wikipedia.org/wiki/Template:Family_tree.  

** Installation **

1.  Just upload convert.php and convert.htm to a web server that has PHP available, which is most web servers.

2. Customize your convert.php and conmvert.htm files.

** Fixed in this version **

1. Lines of relationship no longer behind others unless necessary. 

2. A warning is geenrated if over 80 tiles are used, per the Wikipedia family tree  display limit: http://en.wikipedia.org/wiki/Template:Family_tree.

3. Crossing lines and lines of relkatedness now conform better to the template mentioned in change #2.
